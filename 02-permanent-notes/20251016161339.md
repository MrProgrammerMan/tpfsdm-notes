# Nix expressions
Any piece of nix code is a nix expression.
The following are all nix expressions:
1. 
```nix
1+2
```
2.
```nix
let
 x = 1;
 y = 2;
in x + y
```
3.
```nix
{
	a = 3;
    b = "Hello";
    c = true;
}
```
4.
```nix
rec {
  one = 1;
  two = one + 1;
  three = two + 1;
}
```
5.
```nix
{ a, b, ... }@args: a + b + args.c
```
