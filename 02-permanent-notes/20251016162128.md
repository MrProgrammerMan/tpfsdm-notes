# Nix values
Nix values are the result of evaluating(or running) a [nix expression]([[20251016161339]]).
The values of some nix expressions:
1.
Expression:
```nix
1+2
```
Value:
```nix
3
```
2.
Expression:
```nix
let
 x = 1;
 y = 2;
in x + y
```
Value:
```nix
3
```
3.
Expression:
```nix
{
	a = 3;
    b = "Hello";
    c = true;
}
```
Value:
```nix
{
	a = 3;
    b = "Hello";
    c = true;
}
```
4.
Expression:
```nix
rec {
  one = 1;
  two = one + 1;
  three = two + 1;
}
```
Value:
```nix
{
	one = 1;
    three = 3;
    two = 2;
}
```
5.
Expression:
```nix
{ a, b, ... }@args: a + b + args.c
```
Value:
```nix
<LAMBDA>
```
